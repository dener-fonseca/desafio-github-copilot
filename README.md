# 🚀 desafio-github-copilot

Repositório criado para o **desafio prático do curso _Produtos do GitHub_ da DIO (Digital Innovation One)**.

O objetivo do desafio foi utilizar o **GitHub Copilot** ou alguma **Inteligência Artificial** para auxiliar na criação de pequenos scripts em **Python**, demonstrando operações básicas da linguagem.

---

# 🎯 Objetivo

Praticar o uso de ferramentas de **IA aplicadas ao desenvolvimento de software**, utilizando o **GitHub Copilot** como apoio na escrita de código.

Durante o desafio foram criados **3 scripts em Python** abordando operações fundamentais da linguagem:

- Concatenação
- Repetição
- Cálculo

---

# 🐍 Estrutura do Projeto

```
desafio-github-copilot/
│
├── main.py
└── src/
    ├── __init__.py
    ├── concatenation.py
    ├── repetition.py
    └── calculation.py
```

---

# 📌 Descrição dos Arquivos

## main.py
Arquivo principal responsável por executar o programa e integrar as funcionalidades desenvolvidas nos módulos dentro da pasta `src`.

## src/__init__.py
Arquivo que define a pasta `src` como um **pacote Python**, permitindo a organização e importação dos módulos do projeto.

## src/concatenation.py
Módulo responsável por implementar funcionalidades relacionadas à **concatenação de dados**.

## src/repetition.py
Módulo responsável por demonstrar operações de **repetição**.

## src/calculation.py
Módulo responsável por realizar **operações de cálculo**.

# ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/desafio-github-copilot.git
```

2. Acesse a pasta do projeto:

```bash
cd desafio-github-copilot
```

3. Execute o arquivo principal:

```bash
python main.py
```
